Passo a passo para rodar o projeto:
É necessário instalar o pgadmin e postgresql, após instalar eles iniciar o git e clonar o repositório, abre a pasta do backend do projeto no terminal,
Rode os seguintes comandos "npx prisma generate" e "npx prisma db pull" para carregar o banco que vai vim zerado.
Após carregar o banco rode "npm install" e "npm start" no terminal da pasta do backend e depois no terminal da pasta do projeto no frontend.
